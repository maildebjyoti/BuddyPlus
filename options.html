Buddy Plus!

This is an extension which assists you while typing by replacing some predefined texts. This is a similar implementation of the "Text Replacement" feature in iOS which is missing in Windows. This extension should allow you to complete your mundane tasks quickly.

This extension will only work on input text fields and text-areas. This may ot work with WYSIWUG editors.Go give this a try and explore new world of possibilities in your work flow.

Usage Example:
See ya soon, tc! --> See ya soon, take care!
Please approve by $today. --> Please approve by 29-Feb-2024.

Phrase Replace text    

Note: Only few special characters are allowed. Also the phrases should not start with number or contain spaces in between.

Allowed Characters: !@#$%^&_+-.?
Inbuilt-Function: $today $now $tstamp
